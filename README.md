🌦️ Weather App

A modern Weather App that allows users to check real-time weather conditions for any city worldwide.



🚀 Live Demo
    https://adityakumar88813-glitch.github.io/Weather--App/



📌 Features

- 🔍 Search weather by city name
- 🌡️ Displays temperature, humidity, and wind speed
- 🌤️ Shows real-time weather conditions
- 📱 Responsive design (mobile & desktop)



🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Weather API



⚙️ How It Works

This app fetches real-time weather data from an API and displays temperature, humidity, wind speed, and weather conditions in a simple UI.



🔑 API Used

- OpenWeatherMap API
